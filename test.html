Operating System

OS is a system software which acts as an interface between hardware components and user.

Types of Operating System

  • Batch OS
  • Multitasking OS

DBMS

Database management system is a software which is used to store and retrieve the user's data.

Types of DBMS System

  • Hierarchical database
  • Network database
  • Object Oriented database
